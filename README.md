# Jayanta Nath Portfolio

This workspace now contains a static portfolio website built from Jayanta Nath's public GitHub profile and repositories.

## Structure

- `index.html`: semantic page structure with Hero, About, Skills, Projects, Experience, and Contact sections.
- `styles.css`: visual system, layout, responsiveness, and motion.
- `script.js`: lightweight reveal-on-scroll behavior and footer year.

## Content Strategy

The site is positioned for recruiters and hiring managers who want quick evidence of execution, technical breadth, and project relevance.

- Hero: strong positioning statement plus visible proof points.
- About: concise narrative focused on practical outcomes.
- Skills: grouped by recruiter-friendly categories rather than a long unsorted list.
- Projects: each card includes the technology used, the user or business value, and a clear link to source code.
- Experience: framed around current work, public building, and academic trajectory without inventing formal roles.
- Contact: routes traffic to active public profiles.

## Design Direction

- Visual style: warm editorial layout with glassmorphism panels, soft grid texture, and orange-teal accents.
- Typography: `Space Grotesk` for headings and `Manrope` for body copy.
- Interaction: subtle card lift, smooth anchor navigation, and reveal-on-scroll animation.
- Mobile behavior: stacked layout, full-width buttons, simplified navigation spacing.
